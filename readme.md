## 💻 Getting started

### Requirements

**Clone the project and access the folder**

```bash
$ git clone https://github.com/joao-garcia404/ignite-timer && cd ignite-timer
```

**Follow the steps below**

```bash
# Install the dependencies
$ yarn

# Start the client
$ yarn dev
```
